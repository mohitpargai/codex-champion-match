# Codex Champion Match

Premium interactive memory match game built with React, Tailwind CSS, Framer Motion, and Howler.js.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.

## Deploy Globally With GitHub Pages

1. Create a new GitHub repository.
2. Upload or push this project to that repository.
3. In GitHub, open `Settings > Pages`.
4. Under `Build and deployment`, choose `GitHub Actions`.
5. Push to the `main` branch.
6. Open the `Actions` tab and wait for `Deploy to GitHub Pages` to finish.

Your global link will look like:

```text
https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/
```

For example:

```text
https://mohit.github.io/codex-champion-match/
```
